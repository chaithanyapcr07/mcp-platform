# Support And On-Call

## AI Platform Support

AI Platform supports gateway availability, registry APIs, RBAC/policy decisions, audit export, observability dashboards, SDKs, templates, platform-owned connectors, and the onboarding agent.

## Connector Owner Support

Connector-owning teams support remote connector runtimes, upstream API changes, tool correctness, connector documentation, connector-specific SLOs, and production issues caused by their connector implementation.

## Security Support

Security reviews data classification, secret references, high-risk write actions, approval policy, and SIEM audit requirements.

## Escalation

1. Project team confirms request_id and trace_id.
2. AI Platform checks gateway, policy, and audit.
3. Connector owner checks connector runtime and upstream system.
4. Security reviews denied high-risk or restricted data workflows.
