# Servicenow MCP Connector

This is a generated connector repo owned by service-management-platform. It is not a platform-owned common connector unless AI Platform explicitly adopts it.

## Local Development

```bash
cp .env.example .env
npm install
npm test
npm run dev
```

## Tools

- servicenow.search_incidents
- servicenow.get_incident
- servicenow.create_incident
- servicenow.update_incident

## Register

Submit `registration-request.yaml`, `connector.yaml`, `policy.yaml`, and `validation-report.md` to platform/security review.
