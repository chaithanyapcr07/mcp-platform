# Generated Requests

This folder is for local self-service request artifacts generated by onboarding commands.

These files are examples of intake packages that would normally become a PR, ticket, or platform/security review package.

## Generate An Access Request

```bash
npm run onboard:access -- --connector jira --project ai-platform-demo --tools jira.search_issues
```

## Generate A New Connector Request

```bash
npm run onboard:connector -- --system servicenow --owner-team service-management-platform --mode new-repo
```

## What Belongs Here

- request YAML
- project/tool permission request
- connector registration request
- generated SDD artifacts
- validation report

Do not store real credentials, OAuth tokens, production approvals, or live audit data here.
